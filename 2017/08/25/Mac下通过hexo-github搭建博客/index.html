Mac下通过hexo+github搭建博客

Posted on 2017-08-25

hexo的安装

由于hexo是基于nodejs的，安装前要先安装nodejs，Mac下的安装还是很简单的，打开nodejs官网
node下载下载安装，一路下一步即可。

node安装好之后，打开终端输入node -v 可以看到版本号输出，npm -v可以看到npm的版本号，输出都正常那就没毛病。

终端执行npm -install -g hexo-cli，可能会报错，输出如下图

这个时候还是得在命令前面加个sudo执行就可以了，会叫你输入用户密码

建立GitHub仓库

登录GitHub之后，新建一个username.github.io的仓库，username和GitHub的登录用户名一致。

初始化本地库

执行hexo g 生成文件
hexo s 发布服务

Hexo is running at http://localhost:4000. Press Ctrl+C to stop.
